vkontakte-music-download
========================

Download music from Vkontakte via Python VK API

### Requirements ###
You'll have to install vkontakte from Pypi:

    sudo apt-get install python-pip
    sudo pip install vkontakte pyvkoauth
    
### Usage ###
Edit config.ini file

Launch by typing in your console

    ./audio.py
    python audio.py

Download from group songs:

    ./audio.py 54935444

Download from group wall:

    ./audio.py -w 54935444
